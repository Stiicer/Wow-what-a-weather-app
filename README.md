# Weather Application for the Weather!
Find out if its hot or cold in your local area or abroad!
## Deployed Link

https://stiicer.github.io/Wow-what-a-weather-app/
 
## Description

Allows for a user to search for a city and find the current weather. It will also display a 5 day forecast. Past searches become avaliable options to search.
